Código em Python

A tag CODE por sua vez não organiza os códigos com indetação, que é arrumação padrão do código, observe o exemplo abaixo:

num=int(input('digite um número')) if num % 2 == 0: print(f'0 número {num} é PAR') else: print(f'0 o número {num} é ÍMPAR') print('fim do programa')

A solução pra isso é envelopar essa tag CODE em uma Tag PRE assim o código vai permanecer com a arrumação padrão, mas atenção, pois essa tag considera os espaços e até mesmo as quebras de linha, observe abaixo: 

                
num=int(input('digite um número')) 
if num % 2 == 0:
print(f'0 número {num} é PAR') 
else:
print(f'0 o número {num} é ÍMPAR')
print('fim do programa')
